参考书资源

链接：https://pan.baidu.com/s/1Fc97VROtQv2wBrZD94iCMA 密码：2fp5

我们当初推荐了三本参考书，Head First Java, An Introduction to Java Programming, Java in a Nutshell,我都放在上边的网盘里啦，大家自取呀，不嫌麻烦可以文末通过JiaThis分享出去，让更多的人看到这份资源，也能给我增加一下访问量。

Head First：系列知名度很高，生动形象，有趣，适合看不下去书的同学。

An Introduction to Java Programming：详尽，全面，适合能静心看得下去书的，我感觉这本书很优秀，涉及的知识最全面。

Java in a Nutshell：绝对不推荐预习用，因为里边有些东西讲的不是很严谨（也可能是不合乎主流），适合已经学懂有明辨能力的。

代码

代码可能在验收或者后续思考过程中有改动，所以明辨哦，只起一个参考作用，千万不要抄袭。

唠叨唠叨

大二下算是一个分水岭，期末考完试你会发现曾经不起眼的同学一下子拿了高分成了学霸大佬，因为大二下英方课攻势猛烈，教学模式和之前非常不同，再加上不同口音的全英文授课，会有很多人头痛，也会有很多人斗志激昂得心应手。

所有Prof. Paula组织的编程课程教学模式大概都是这样的：

课上：按照课件讲课，穿插一些实例，她在课上现敲代码。

课下：自己看参考书。

考试范围：课件+参考书+lab+QM上所有的资料，所有全部你能接触到的，不是像以前一样只看课件就能拿高分了啊！！！举个例子，课件里我们讲实例变量instance variable, 在别的书里可能叫data field, object attribute…这些术语指的都是同一个东西，虽然课上我们的叫法是一致而统一的，但出题时就不一定了哦，我们MCQ1就用的data field这个词，很多没看过参考书的同学都不知道这是个啥，所以一定要看书哦；还有object，后边有时候也会用instance这个词，都是一个东西，但不多看不了解就可能纠结半天不懂到底在说什么。
